# 🏀 篮球记分牌 与 ⏱️ 计时器

概述  
本仓库包含两套用于篮球比赛的工程文件（.ms14），分别负责比分显示与比赛计时

主要文件 📦
- 篮球记分牌.ms14 — 记分牌工程，通过电路设计完成计分功能
- Timer.ms14 — 比赛计时器工程，支持倒计时、暂停/恢复与提示功能

环境条件 ⚙️
- 支持 .ms14 的软件（例如 Multisim ）
- 所用软件版本应与工程文件兼容

快速开始 🚀
1. 克隆仓库到本地：
   ```powershell
   git clone <仓库地址>
   ```
2. 在支持 .ms14 的软件中打开相应工程文件，并确认工程设置与目标显示

主要功能 ✨
- 实时比分显示与队伍管理
- 比赛计时（倒计时、暂停、恢复）
